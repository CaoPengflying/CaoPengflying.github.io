hexo+Github搭建个人博客

Posted on 2018-08-27 | In 个人建站

  个人建站：之前自己买了个阿里云的服务器，然后使用wordpress建了一个人的网站，后面服务器过期了，之前的项目也不用自己维护了，所以就没有继续续费，但是也不想在csdn上面去。后面看了一些大牛的建议说把站点建在github上面。确实比较好，GitHub是目前全球最大的开源平台，并且在这里建站支持更改域名。所以自己就捣鼓了一下，在github上面建了 自己的博客。然后把自己建站的过程分享一下，中间踩过很多坑，写的详细一点，尽量帮助其他小白建站。

##hexo+github建站需要用到的软件和账号##

  1. nodejs
  2. git
  3. github账号
  4. hexo

##搭建环境##
1 安装nodejs

  安装nodejs大家可以去我的博客-Node和express环境搭建去学习安装，也可以去菜鸟网站去学习安装

2 安装git

  安装git可以去stormzhang的博客或者廖雪峰的博客

3安装hexo

  安装好了nodejs之后，就可以使用npm去安装hexo了。首先在电脑的软件安装位置右击选择Git Bash Here
并输入命令npm install -g hexo执行命令后，该目录下会多出来一些modules这些都是hexo中用到的。如图是我执行命令后多出来的文件

然后我再E盘目录下新建了一个hexo的文件夹，并在这个文件夹执行命令 hexo init
注意： hexo必须是空的文件夹，否则执行init命令会报错。我原意是在E/MyGitHubBlog目录下安装的但是执行了 安装命令后就多出了文件所以不能再执行hexo init命令了
再执行npm install
然后再执行hexo server
如果看到如图显示，则hexo安装成功

注意：hexo默认在4000端口 如果端口被占用的错误，需要查找端口并终止进程，详情见博客端口被占用
安装成功后，可以打开浏览器输入地址http://localhost:8080可在浏览器查看到hexo的首页地址，他默认给我们写了一篇博客

4注册github账号，并创建仓库

  接下来要将博客推送到github上面。github提供了一个pages的功能可以让用户搭建自己的博客。没有账号需要注册一个账号，有了账号则直接登录。登录后创建一个repository

然后填写repository name就行了，要注意的是：这个名字的格式必须为youname.github.io，并且必须和你的账户名相同，CaoPengflying，我的地址就是CaoPengflying.github.io
