hexo+github搭建个人博客更改域名

发表于 2018-04-29 | 分类于 个人建站 | 阅读次数:

##购买域名

  推荐在阿里云购买域名
.com的域名比较商业化，也比较专业。当然这个域名会比较贵。如果是只有博客使用的话，无所谓什么域名，都可以。但是会毕竟是个性化一点的东西。也不差这个钱，自己注册一个个性的域名比较好

##配置域名

买好了域名之后登录到自己的控制后台，可以找到域名。买完域名之后是需要备案的。备案是有地区的。选择一个广州或者深圳的域名可能会备案速度快一些。耐心等待即可，大概需要2周时间。
接下来就是解析这个域名，会进入到如下图的页面

如果是第一次使用，那就选择新手引导。当然也可以选择添加记录
下面的记录就是我的github个人博客修改后的域名

点击添加记录后如以下界面

记录类型 记录类型可以选用A-就可以
主机记录 主机记录，可以使用www，如果是个人博客也可以用blog
解析线路 采用默认即可
记录值 这个是ip地址，要输入你github上的ip地址
TTL 采用默认即可



如何获得github上面的ip地址呢，打开命令窗口输入

username是你之前搭建好的github博客，这里就可以获得ip地址，将这个地址放到刚刚的记录值中就可以

##在github上配置自己的域名
打开自己在github上新建的reposity ，里面有一个设置按钮，点开设置按钮，如图所示

第一个框是新建的reposity ，下一个框就是设置按钮，点击设置后进入到下个界面

在这里输入你的个人域名即可。
