hexo+github搭建个人博客更改主题

发表于 2018-04-29 | 分类于 个人建站 | 阅读次数:

hexo搭建个人博客后，这个风格非常难看，现在比较好看的主题就是next主题，简约好看。接下来告诉大家，hexo更换主题以及next主题样式修改步骤

##hexo修改为next主题

  next主题在github上面维护。大家可以下载这个主题，并将这个主题放在hexo安装的themes目录下 如我的主题下载放在如图所示

这个主题的下载可以登录github 搜索 hexo-theme-next 地址为 hexo next主题点击下载。当然也可以采用 git命令直接下载

如果是压缩包的话要记得解压。然后在hexo的站点配置文件中将主题更改为刚刚下载的next文件夹的名字 我的名字叫做theme-next那么就修改站点的配置文件_config.yml使用文本工具打开，然后搜索theme如图位置，将后面的值改为自己下载的文件的名字。

##修改样式

  • 改为中文版本
      首先需要在站点的配置文件中，将语言改为中文打开_config.yml配置文件，然后查找language，将language改为 zh-Hans就是中文了，然后之后的导航页面，打赏 分享等都是中文了

  • 在每篇文章末尾统一添加“本文结束”标记

  • 隐藏网页底部powered By Hexo / 强力驱动

  • 添加博文点击量功能

  • 页面点击采用小红心模式
  • 博文压缩
  • 添加打赏

可直接下载修改过后的next主题
