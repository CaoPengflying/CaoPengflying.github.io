File /hello/input.data._COPYING_ could only be replicated to 0 nodes instead of minReplication (=1).

发表于 2018-05-05 | 分类于 hadoop | 阅读次数:

错误详情：向hdfs文件系统上传文件失败

提到了here are 0 datanode(s) running and no node(s) are excluded in this operation.
然后输入jps命令查看进程，发现只有 jps、namenode两个进程，没有启动从节点
再执行命令 sbin/hadoop-daemon.sh start datanode
再重复执行put命令成功
