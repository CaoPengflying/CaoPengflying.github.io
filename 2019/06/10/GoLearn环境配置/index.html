Go环境配置

下载

Go下载

windows 安装包类型有两种，1. msi 安装版 2. zip 压缩版

采用Msi安装选择安装路径，建议不要有中文的路径，防止后续不识别中文。

我的安装路径为E:\Go

环境变量配置

  1. GOROOT

    配置GOROOT可以在全局执行go命令，需要将其配置到系统变量PATH中

    添加变量 GOROOT 值为Go安装的根目录 E:\Go

    并在Path后增加 %GOROOT%\bin

  2. GOPATH

    配置gopath的目的是要下载第三方依赖，如果不配置GOPATH可能会提示没有环境变量GOPATH,或者第三方依赖会放在c盘用户下，不方便管理，建议配置GOPATH，目录不要为GOROOT的目录

开发工具选择

建议使用Goland，Jetbrains系列产品，破解方法

在Goland配置GoRoot和GoPath，GoPath可以采用全局的也可以采用当前项目的。配置路径 file -> setting -> go
